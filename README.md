# ?? RD DevOpsKit

**Rubber Duck's DevOpsKit** is a personal, all-in-one utility toolbox for DevOps and sysadmin tasks, built with classic .NET WinForms.  
Created by and for [RubberDuck01](https://github.com/RubberDuck01), it’s designed to streamline everyday operations, troubleshooting, and automation workflows.

---

## ?? Features

- ?? **Text Diff Tool** – Quickly compare two text snippets or files
- ?? **Base64 Encoder/Decoder** – Encode and decode Base64 strings
- ?? **Kubernetes YAML Verifier** – Validate and lint K8s YAML manifests
- ? **More to come!** – I update as I go...
